選擇公文範本

⚙️ 設定

設定公開的靜態檔案來源網址，系統啟動時會自動下載最新的公文範本、議員名單及組別資料。
不需要任何 API 金鑰或帳號驗證。

支援 github.com/…/tree/… 或 raw.githubusercontent.com 格式，完全免費且無需金鑰。
📁 資料夾內需包含以下檔案：
  • 所有 .di 副檔名的檔案都會自動視為公文範本
  • 議員分組.csv — 議員名單及組別（格式如下）
議員分組.csv 格式範例 
警政衛生部門,1,洪健益
警政衛生部門,1,劉耀仁
警政衛生部門,2,應曉薇
警政衛生部門,2,耿葳

格式：部門, 組別號碼, 姓名（無標題列，逗號分隔）
