選擇公文範本

⚙️ 設定

設定公開的 Google Drive 資料夾，系統啟動時會自動下載最新的公文範本、議員名單及組別資料。

從 Drive 資料夾的分享連結中複製 ID（URL 中 /folders/ 之後的部分）
至 Google Cloud Console 啟用 Drive API 後建立 API 金鑰（限制為 HTTP 參照者網址以提高安全性）
Drive 資料夾結構：
  • 8-1_函復議會協調案件.di — 公文範本（命名格式：案件類型_名稱.di）
  • legislators.csv — 議員名單（每行一個姓名）
  • groups.csv — 組別對應表（欄位：組別,議員姓名）
